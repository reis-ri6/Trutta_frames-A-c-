# Trutta Hub

Коротко:
- Що таке Trutta Hub.
- Які основні модулі (Trutta, TJM, ABC, domains).
- Для кого цей репозиторій.

## Структура
- `/docs` — продуктова документація.
- `/concepts` — моделі й фреймворки.
- `/domains` — індустріальні домени.
- `/schemas` — фізичні схеми даних.
- `/templates` — темплейти проектів/дизайну/архітектур.
- `/agents` — агенти та агентські системи.
- `/progress` — статуси артефактів та roadmap.
- `/ingestion` — правила та трансформації для repo-ingestion-agent.
- `/import` — сирі матеріали для інгесту (raw/legacy/unknown).

## Швидкий старт
1. Клонувати репо.
2. Прочитати `docs/guides/getting-started-trutta.md`.
3. Використати один із темплейтів у `/templates/projects`.
